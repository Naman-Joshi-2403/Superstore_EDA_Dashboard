# 🛒 Superstore EDA Dashboard

An end-to-end Exploratory Data Analysis project on the Kaggle Global Superstore dataset, deployed as a live interactive dashboard on Render.

## 🚀 Live Demo
> `https://superstore-eda.onrender.com` *(update after deployment)*

---

## 📁 Project Structure

```
superstore_eda/
├── app.py                        # Streamlit dashboard entry point
├── config.py                     # All constants and paths (nothing hardcoded)
├── requirements.txt
├── Dockerfile
├── data/
│   ├── raw/superstore.csv        # Original data (never modified)
│   └── processed/superstore_clean.csv  # Generated by pipeline
├── src/
│   ├── data_loader.py            # Load + schema validation
│   ├── data_cleaner.py           # Full cleaning pipeline
│   ├── feature_engineer.py       # Derived features
│   └── visualizer.py             # All Plotly chart functions
└── notebooks/
    └── eda_exploration.ipynb     # Scratch exploration
```

## ⚙️ Local Setup

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/superstore-eda.git
cd superstore-eda

# 2. Create virtual environment
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Place the raw data
# Copy superstore.csv → data/raw/superstore.csv

# 5. Run the dashboard
streamlit run app.py
# Open http://localhost:8501
```

## ☁️ Deploy to Render

1. Push this repo to GitHub
2. Go to [render.com](https://render.com) → New Web Service
3. Connect your GitHub repo
4. Set Environment: **Docker**, Branch: **main**
5. Click **Create Web Service** — live in ~5 minutes

## 📊 Dashboard Features

| Tab | Content |
|---|---|
| 📈 Sales Trends | Monthly trend, YoY growth, quarterly profit |
| 💰 Profitability | Sub-category diverging bar, discount impact |
| 🗺️ Regional | US choropleth, region×category heatmap |
| 👥 Customers | Segment pie, top N customers, margin by segment |
| 📦 Operations | Shipping days boxplot, ship mode profit |
| 🔬 Deep Dive | Distributions, correlation heatmap, raw data + CSV export |

---

## 🔑 Key Insights

- **Tables** sub-category has the largest losses — driven by high discounts (>45%)
- **Technology** has the highest profit margin despite moderate sales volume
- **Q4** (Oct–Dec) consistently shows peak sales across all years
- **High discounts (>40%)** correlate with ~65% loss rate on order lines
- **California** and **New York** account for ~35% of total revenue
